UiToAi
Usage Guide

Quick Start

  1. Open any webpage you want to replicate (e.g. a marketing site or web app).
  2. Click the UiToAi extension icon to open the popup.
  3. Click Start Run to begin a collection session.
  4. Click Capture Snapshot to extract design tokens, layout rules, motion and a11y hints.
  5. Optional: click Pick Element to enter element selection mode.
  6. Click Export for AI to export files into a folder: UiToAi-export-{host}-{timestamp}/
    • ai/spec.json - source of truth data
    • ai/prompt.md - prompt guidance for AI coding tools
    • samples/elements.json - element samples (if any)
  7. Click Stop Run to finish the session.

快速开始

  1. 打开你要复刻的网页（营销页 / Web 应用 / 文档页等）。
  2. 点击 UiToAi 扩展图标打开弹窗。
  3. 点击 Start Run 开始本次采集。
  4. 点击 Capture Snapshot 提取设计 tokens、布局规则、动效与可访问性线索。
  5. 可选：点击 Pick Element 进入元素选择模式。
  6. 点击 Export for AI 导出到文件夹： UiToAi-export-{host}-{timestamp}/
    • ai/spec.json：规范数据（事实来源）
    • ai/prompt.md：给 AI 工具的提示词模板
    • samples/elements.json：元素样本（如果有）
  7. 点击 Stop Run 结束采集。

Troubleshooting

  • Popup buttons disabled: Make sure the current tab is a normal web page (not edge://, chrome://, extension pages).
  • Export blocked: Your browser may block multiple downloads. Allow downloads for this extension and retry.
  • How to view logs: Go to edge://extensions / chrome://extensions → extension Details → Service worker → Inspect.
  • 弹窗按钮不可用： 请确认当前标签页是普通网页（不是 edge://、chrome://、扩展页等）。
  • 导出被拦截： 浏览器可能会阻止批量下载。请允许此扩展下载多个文件后重试。
  • 如何查看日志： 打开 edge://extensions / chrome://extensions → 详细信息 → Service worker → Inspect。
